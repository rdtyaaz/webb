6. Description List (Daftar Deskripsi)

HTML
HyperText Markup LanguageHyper (HTML) adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. HTML menggunakan elemen-elemen seperti heading, paragraf, gambar, tautan, tabel, dan daftar untuk menyusun konten di browser.
CSS
Cascading Style Sheets (CSS) adalah bahasa script yang digunakan untuk mengatur tampilan dan tata letak elemen dalam halaman web yang dibuat dengan HTML.
JavaScript
JavaScript (JS) adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif dan dinamis. Dengan JavaScriptn, kita bisa menambahkan animasi, validasi formulir, menangani event, (klik, hover) dan banyak lagi.
